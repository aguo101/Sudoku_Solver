# Sudoku_Solver
Sudoku solving program I built for learning purposes

To build: `run dune build sudoku.exe`
To run: `./_build/default/sudoku.exe`
        - input puzzle with space seperated digits and 0 for unknown values
        - ie.
0 0 3 0 2 0 6 0 0
9 0 0 3 0 5 0 0 1
0 0 1 8 0 6 4 0 0
0 0 8 1 0 2 9 0 0
7 0 0 0 0 0 0 0 8
0 0 6 7 0 8 2 0 0
0 0 2 6 0 9 5 0 0
8 0 0 2 0 3 0 0 9
0 0 5 0 1 0 3 0 0
